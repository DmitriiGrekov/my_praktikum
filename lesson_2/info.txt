В ООП:
класс - набор свойств и методов, описывающих объекты
объект - экземпляр класса
переменный - свойства
функции - методы
интерфейс - способ, которым вы обращаетесь к свойствам и методам объекта(функция show в пимере)

Инкапсуляция - объединение и скрытие методов и свойств, и предоставление доступа к ним через простой внешний интерфейс.

Полиморфизм - возможность взаимодействовать с объектами разных типов через одинаковые интерфейсы, обращаться к свойствам и методам обшим для всех объектов (у всех наследников класса Bird есть метод show())




GITHUB

git status - чтобы узнать о состояние файлов  в репозитории
git add - команда для добавление файлов в Staging Area, в индекс
git rm --cached <file> вы можете перевести файл в состояние неостлеживаемый

Каждый коммит сохраняет актуальное состояние файлов, и вы можете сравнивать разные состояния от коммита к коммиту

git show - показать изменения, внесенные в оределённом коммите, 
git reset - отказаться от коммита
gir reset HEAD program.py - откатить на один коммит назад в определенном файлe
где HEAD - означаетс самый последний коммит
